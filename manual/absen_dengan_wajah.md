# Panduan Absensi (Dengan Verifikasi Wajah)

Panduan ini ditujukan bagi pengguna (pegawai/siswa) yang diwajibkan melakukan absensi menggunakan verifikasi wajah biometrik (Face Recognition) dan deteksi lokasi (GPS).

## 1. Login ke Aplikasi
1. Buka aplikasi Absensi.
2. Masukkan identitas Anda (misalnya: ID Pegawai, NISN, atau Username) pada kolom yang tersedia.
3. Masukkan kata sandi (password).
4. Tekan tombol **Login**.

## 2. Pendaftaran Wajah (Hanya Dilakukan Sekali)
Jika ini adalah pertama kalinya Anda menggunakan aplikasi, Anda wajib mendaftarkan wajah terlebih dahulu.
1. Setelah login, pada halaman utama (Beranda), Anda akan melihat peringatan berwarna merah **"Wajah Anda belum terdaftar!"**.
2. Pastikan Anda berada di tempat dengan pencahayaan yang cukup.
3. Tekan tombol **Daftarkan Wajah Sekarang**.
4. Kamera depan akan terbuka. Arahkan wajah Anda ke dalam bingkai kamera dan pastikan wajah terlihat jelas.
5. Sistem akan memindai kontur wajah Anda. Jika berhasil, sistem akan otomatis menyimpan data biometrik Anda. Kini Anda siap untuk melakukan absensi.

## 3. Cara Absen Masuk
1. Buka aplikasi dan masuk ke tab **Beranda** (ikon rumah).
2. Perhatikan peta lokasi di layar. Pastikan Anda berada di dalam area yang ditentukan (**lingkaran biru** pada peta).
3. Pastikan teks di atas peta berbunyi **"Anda berada di dalam jangkauan"** (warna hijau).
4. Jika sudah sesuai, tombol **Absen Masuk** akan aktif. Tekan tombol tersebut.
5. Kamera akan terbuka. Arahkan wajah Anda kembali ke kamera untuk proses pemindaian.
6. Setelah wajah terverifikasi cocok, aplikasi akan mengirimkan data absensi Anda ke server secara otomatis.
7. Jika sukses, kotak peringatan di Beranda akan berubah warna menjadi hijau dengan tulisan "Anda sudah absen hari ini".

## 4. Cara Absen Pulang
1. Saat jadwal pulang tiba, buka kembali aplikasi.
2. Pastikan GPS Anda terbaca dan Anda masih berada di dalam area kantor/sekolah (lingkaran biru).
3. Pada layar Beranda, tekan tombol **Absen Pulang**.
4. Anda akan diminta untuk melakukan pemindaian wajah sekali lagi.
5. Setelah berhasil terverifikasi, jam kepulangan Anda akan tercatat di sistem.
6. Selesai! Anda dapat melihat rincian jam masuk dan pulang di tab **History** (ikon jam di menu bawah).
